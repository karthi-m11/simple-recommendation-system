# Simple Recommendation System

## Project Description
This project implements a simple recommendation system that suggests
items to users based on their preferences. It follows a content-based
filtering approach, where user-selected genres are matched with item
categories to generate recommendations.

This project was completed as Task 4 of my internship to understand
the basic working of recommendation systems.

## How It Works
- The user enters preferred genres
- The system compares preferences with item categories
- Matching items are recommended to the user
